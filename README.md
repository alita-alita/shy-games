# SHY Games

Two lightweight, ad-free browser games for children.

## Games

- Bubble 2048: `index.html`
- Kids Learning: `kids-learning/index.html`

## Local Preview

```sh
python3 -m http.server 8765
```

Then open:

- `http://127.0.0.1:8765/`
- `http://127.0.0.1:8765/kids-learning/`

## GitHub Pages

After deployment, the public URLs are expected to be:

- `https://alita-alita.github.io/shy-games/`
- `https://alita-alita.github.io/shy-games/kids-learning/`
